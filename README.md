# UNIARI — Tela de Login Institucional
Interface de login para o portal acadêmico da Universidade Ari de Sá.

# SOBRE O PROJETO

Este projeto é uma refatoração da tela de login do portal acadêmico da UNIARI (Universidade Ari de Sá). A proposta foi modernizar a interface, tornando-a mais limpa, responsiva e intuitiva para alunos e professores.

A versão atual é um **protótipo funcional** com validação de formulário e feedback visual, sem integração com banco de dados — ideal para apresentar a estrutura e o design antes da implementação completa.

# TECNOLOGIAS UTILIZADAS

- **HTML5:** Estrutura semântica e acessível
- **CSS3:** Estilização com gradientes, flexbox e responsividade
- **JavaScript:** Validação de formulário, alternância de abas e notificações toast
- **Google Fonts:** Tipografia (Encode Sans Expanded)

# FUNCIONALIDADES

- Design responsivo (celular, tablet e desktop)
- Alternância entre abas "Aluno" e "Professor"
- Validação de e-mail institucional (@fas.edu.br)
- Validação de senha (mínimo 12 caracteres)
- Notificações toast com:
  - Mensagens de erro personalizadas
  - Animação de entrada (slide)
  - Cronômetro visual (barra de progresso)
- Botão "Esqueci minha senha" (placeholder)
- Layout em duas colunas (formulário + gradiente decorativo)
- Logo institucional no topo
